### US-001: Basic Calculator Functionality

As a user, I want to perform basic arithmetic operations so that I can quickly calculate simple math problems.

**AC-US1-001:** The calculator should display a clear interface with number buttons (0-9), operator buttons (+, -, ×, ÷), and equals button
**AC-US1-002:** The calculator should perform addition, subtraction, multiplication, and division operations correctly
**AC-US1-003:** The calculator should handle decimal point input and display results with appropriate decimal precision

### US-002: Calculation History Tracking

As a user, I want to view my previous calculations so that I can reference past results without re-entering them.

**AC-US2-001:** The calculator should store the last 10 calculations in a history panel
**AC-US2-002:** Each history entry should display the full calculation (e.g., "5 + 3 = 8") and timestamp
**AC-US2-003:** The history panel should be collapsible to save screen space
